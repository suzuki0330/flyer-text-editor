# チラシ文字エディタ

チラシ・ポスター画像の文字を自動認識して、レイアウトを保ったまま編集できるテキストに置き換えるWebアプリです。

**使い方**: https://suzuki0330.github.io/flyer-text-editor/ を開き、画像をドラッグ＆ドロップするだけ。

- 文字認識(OCR)も画像処理も、すべてお使いのブラウザの中だけで行われます。画像が外部のサーバーに送信されることはありません。
- 単色・グラデーション背景の上の文字をテキスト化します。写真や飾りの上の文字は、背景を壊さないため画像のまま残ります。
- 編集後は「PNGで保存」でダウンロードできます。
